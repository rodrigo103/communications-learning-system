Para que Notion reconozca correctamente el Markdown, prueba estas soluciones:

## Solución 1: Pegar como Markdown (más fácil)
1. Copia el texto Markdown
2. En Notion, escribe `/` para abrir el menú
3. Busca "Import" o "Importar"
4. Selecciona "Markdown & CSV"
5. Pega o sube el archivo

## Solución 2: Usar atajos de Notion
Notion usa Markdown mientras escribes:
- `# texto` → Heading 1
- `## texto` → Heading 2  
- `**texto**` → **negrita**
- `*texto*` → *cursiva*
- `- texto` → lista

Pero tienes que presionar **Espacio** después del símbolo para que se convierta.

## Solución 3: Convertir a HTML primero
Si el Markdown es complejo, puedes:
1. Usar un conversor online (markdowntohtml.com)
2. Copiar el HTML resultante
3. Pegarlo en Notion (Ctrl+Shift+V en Windows, Cmd+Shift+V en Mac)

## Solución 4: Guardar como archivo .md
1. Guarda tu contenido como archivo `.md`
2. En Notion: "Import" → "Markdown & CSV"
3. Selecciona el archivo
